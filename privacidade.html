Política de Privacidade - BRX STORE BOT

Última atualização: 28/06/2025

1️⃣ Dados Coletados

O bot coleta apenas o necessário:

  • IDs de usuários do Discord;
  • IDs de guild (servidores);
  • Informações sobre pedidos realizados via comandos;
  • Logs de ações como banimentos, alterações de mensagens e verificação de usuários.

2️⃣ Uso dos Dados

Os dados são usados exclusivamente para garantir a verificação de usuários, processar pedidos na loja virtual, gerar relatórios de logs para moderação e manter a segurança do servidor.

3️⃣ Compartilhamento

Os dados não são compartilhados com terceiros, exceto se exigido por lei ou ordem judicial.

4️⃣ Segurança

Os dados ficam armazenados em um banco de dados local (SQLite) controlado pelo administrador da BRX STORE. São adotadas boas práticas para impedir acesso não autorizado.

5️⃣ Direitos dos Usuários

Você pode solicitar a exclusão dos seus dados entrando em contato com o responsável via [seu e-mail de suporte ou Discord].

6️⃣ Alterações

Podemos atualizar esta Política. Notificações sobre alterações relevantes podem ser feitas no servidor Discord.
